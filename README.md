# Project02-SpaceShooter

## Context and Date
This project was created as Project 02 for a game development course.  
Date: February 2026

## Description
Project02-SpaceShooter is a 2D arcade-style space shooter game (beginner friendly).
The player controls a spaceship using the keyboard and can fire a weapon to destroy enemies.  
The goal is to survive, defeat enemies, and earn points.

The game includes a start menu, gameplay scene, and an end-game screen when the player loses.

## Implementation
- Player controlled spaceship using keyboard input (WASD, SPACEBAR)
- Player can fire bullets out of the ship.  
- Collision detection using 2D colliders  
- Scoring system that increases when enemies are destroyed  
- Start game menu  
- End-game screen (game over state)
